# Weather Dashboard (React + Vite)

Цей проєкт є інтерактивним погодним дашбордом, створеним для відстеження поточної погоди та прогнозів на 24 години і 7 днів. 

## Технології
- **React JS** для компонентного підходу.
- **Vite JS** для швидкої збірки.
- **Zod** для валідації відповідей API.
- **Chart.js (react-chartjs-2)** для красивих графіків.
- **Open-Meteo API** (Geocoding & Forecast).

## Запуск локально
1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть dev-сервер:
   ```bash
   npm run dev
   ```

## Особливості
- "Glassmorphism" UI зі змінними фонами (динамічно залежать від погоди).
- Автодоповнення міст.
- Кнопка автовизначення локації.
